《晴空雜貨店》品牌電商專案
從品牌經營、數位轉型到資安防禦的實戰全紀錄

D2C Brand Project

晴空雜貨店 (Clearsky Tuckshop)

品牌理念：晴空萬里下的復古雜貨店

本專案源於對懷舊情懷的熱愛，將家鄉傳統食品產業「海鮮罐頭」搬上數位平台，
從 2023 年啟動至今，完整經歷了從雲端 VM 佈署、資安攻防實戰到最終「靜態化架構」遷移的品牌全生命週期。

技術與營運演進歷程

第一階段：傳統產業轉型與雲端佈署 (2023)

利用 GCP (Google Cloud Platform) 建構動態電商環境，成功將家鄉實體供貨鏈對接到線上銷售端。

  • 基礎設施：負責 GCP Compute Engine、網域採購、DNS 指向與伺服器環境初始化。
  • 數位轉型：重新定義傳統產品包裝，建立 D2C 自有品牌視覺，並取得年均 1,500+ 活躍使用者流量。
第二階段：資安攻防實戰 (2023)

上線初期遭遇駭客腳本攻擊，在無 AI 輔助下，靠著自主研究與技術諮詢，成功完成災難復原。

  • 資安強化：獨立分析 Server Log、修復系統漏洞，並將資安評等提升至 SSL Labs B 級。
  • 韌性建立：透過 Firewall Rules 與身分驗證優化，確保遭遇攻擊後仍能維持網站穩定運行。
第三階段：靜態化遷移與永續展示 (2026.03)

因應供貨鏈波動與維護成本，決定將網站轉型為 Jamstack 靜態架構，作為數位作品集永久留存。

  • 技術架構：轉譯 5,000+ 動態頁面為靜態 HTML，並託管於 GitHub Pages。
  • 效益優化：實現零成本主機支出，同時完全免除 SQL Injection 等資安風險，提升載入效能。

專業成果亮點

  • 自有品牌經營：具備從產品規劃、行銷推廣到技術維運的品牌主思維。
  • 資安防禦實績：具備處理真實中毒事件與系統加固的 Incident Response 經驗。
  • 流量轉化能力：營運期間成功創造 1,200+ 次 Organic Search 點擊，具備 SEO 基礎實力。
